# Laptop Forensic Utility

## Overview

This is a simple Windows-based tool for scanning laptops and collecting forensic data like files, system activity, and user actions.

## Features

* Scan system for files and activity
* Detect deleted files
* Check browser history
* Track USB usage
* View system logs
* Generate reports

## Supported OS

* Windows 10 / Windows 11

## Usage

1. Install the tool
2. Run as Administrator
3. Start scan
4. View results

## Note

Use only on authorized systems. Do not modify data during scanning.
